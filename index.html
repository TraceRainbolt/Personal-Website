    LC3 Simulator

    Python, PyQt, Numpy

    In my into to computer architecture class we used a hypothetical computer named the LC3 (Little Computer 3). However, I felt as if the default simulator left a lot to be desired.

    Using Python 2.7 I built my own with the features I wanted, including loading multiple files at once, a history of memory locations you've visited, and a cleaner and more fluid GUI.
